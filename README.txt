# gpsL80_Nokia5110_Beeper
Presenta les dades recollides del gps L80 al display LCD nokia 5110, a cada actualitzacio del display sona una nota.

Aci tenim la base, ara molaria afegir funcionalitats, com per exemple :

- diferents persentacions de les dades al display. 
  Mitjançant un polsador podriem passar de la pantalla amb totes les dades a un altre pantalla només amb la velocitat actual escrita amb numeros enormes,
  i a d'altres pantalles amb diferent info.

- afegir una SD al sistema.
  Podriem tenir un conta kilometres que guardés registres parcials i totals,
  i/o fins i tot poder guardar un registre del track..
